Socket Io

(redux middleware)

/
notify the server that the user has visited the site
  • request

Connect to server

  • connection adress: (base url)
  • autoConnect: false (required)
  • reconnect: true (required)
  • transports: web-socket (required)
  • route: server/

dispatch connection action

  • type: 'server/init' (required)
  • payload (required) (object)
    • userId (required) (string)

dispatch disconnection action

  • type: 'server/forceDisconnect' (required)
when reconnect, 'server/init' action must be dispatched to the server again

Example:

                                    
    Middleware:
                import createSocketIoMiddleware from 'redux-socket.io';
                import io from 'socket.io-client';

                let socket = io('http://localhost:5000/',{autoConnect:false,reconnection: true,transports: ["websocket"]});
                export const OnlineSocketIoMiddleware = createSocketIoMiddleware(socket, "server/");
                socket.connect()
    redux store:
                 import { createStore, applyMiddleware } from 'redux';
                 import {OnlineSocketIoMiddleware} from "../Redux-middleware/initOnlineSocketMiddleware"

                  let store = applyMiddleware(OnlineSocketIoMiddleware)(createStore)(reducer);
    dispatch:
                 const userId="1232asdf23523rasf23422asdfaww"
                 dispatch({type:'server/init', payload:{userId}})

common chat
  • request response

Connect to chat

  • connection adress: (base url) /chat
  • autoConnect: false (required)
  • reconnect: true (required)
  • transports: web-socket (required)
  • route: chat/

dispatch action for get messages

  • type: 'chat/messages' (required)
  • payload (required) (object)
    • pageNumber (required) (integer)
    • pageSize (required) (integer)

dispatch action for send new messages

  • type: 'chat/newMessage' (required)
  • payload (required) (object)
    • sender (ownerId) (required) (string)
    • text (required) (string)
when reconnect, 'chat/messages' action must be dispatched to the server again

Example:

                                    
    Middleware:
                import createSocketIoMiddleware from 'redux-socket.io';
                import io from 'socket.io-client';

                let socket = io('http://localhost:5000/chat',{autoConnect:false,reconnection: true,transports: ["websocket"]})
                export const ChatSocketIoMiddleware = createSocketIoMiddleware(socket, "chat/")
                socket.connect()
    redux store:
                 import { createStore, applyMiddleware } from 'redux';
                 import {ChatSocketIoMiddleware} from "../Redux-middleware/initChatMiddleware"

                  let store = applyMiddleware(ChatSocketIoMiddleware)(createStore)(reducer);
    dispatch:
                 dispatch({type:'chat/messages', payload})
                 dispatch({type:'chat/newMessage', message})

action for get all messages

  • type: 'chat/getChatMessages' (required)
  • chatMessages (required) (array)

action for get new messages

  • type: 'chat/getNewMessage' (required)
  • newMessage (required) (object)
    • sender (required) (object)
    • text (required) (string)
    • createdAt (required) (string)

Example:

                                    
    extraReducers: {
        ['chat/getChatMessages']:(state,action)=> {
            state.messages = [...state.messages,...action.chatMessages]
        },
        ['chat/getNewMessage']:(state,action)=> {
                state.messages = [action.newMessage,...state.messages]
        }
    }
